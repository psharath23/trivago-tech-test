# TRIVAGO FRONTEND CASE STUDY
## CLIENT SIDE PROJECT

### HOW TO RUN
1. npm i 
2. npm start

### WHAT I HAVE USED
1. React16
2. React.lazy() for code splitting
3. React Hooks for using state in functional components
4. React Router for Routing 
5. SCSS for styling
6. create-react-app for faster building
